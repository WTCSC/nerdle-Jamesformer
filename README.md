# Nerdle math equation guessing game
This program is a math based guessing game with the format and style of the ny times popular game of wordle but insted of words it uses basic mathimatical equations

## Requirements
This projects requires the following to work properly. If you do not have the requirements their could be errors: 
**Python version 3.12 or higher** and
**Git version 2.51 or higher**

## Installation
To install this program you need to run the following command: 
```
git clone (your reppository URL)
```

## examples
this is what the program should look like when run: 
```
============================================================
             WELCOME TO NERDLE!
         A Math Equation Guessing Game
============================================================

GAME RULES:
- Guess the 8-character math equation
- You have 6 attempts to get it right
- Each equation has the format: NN+NN=NN (or -, *, /)
- After each guess, you'll get color-coded feedback:
  *  Green : Correct character in correct position
  *  Yellow : Correct character but wrong position
  *  Gray : Character not in the target equation

EXAMPLE:
Target:   12+34=46
Guess:    15+35=50
Feedback:  1  5  +  3  5  =  5  0 

Valid characters: 0123456789+-*/=
============================================================
Generating your equation...
Equation generated! Start guessing...

(game status and guesses intill game over)

The correct equation was: (insert equation)
```
